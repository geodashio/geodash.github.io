Introduction

GeoDash is a modern web framework and approach for quickly producing visualizations of geospatial data. The name comes from "geospatial dashboard".

The framework is built to be extremely extensible. You can use GeoDash server (an implementation), the front-end framework, backend code, or just the Gulp pipeline. Have fun!

Implementing Projects

GeoDash Server

GeoDash Server is an open source implementation of the GeoDash framework and associated approach.

SPARC

SPARC is a Spatial Risk Calendar that combines data layers covering hazard exposure and population vulnerability to determine baseline probabilistic levels of humaitarian impact associated with particular hazard types, on a monthly basis, at a sub-national level, around the globe.

SPARC is a digital service provided by the World Foood Programme (WFP), Emergency Preparedness and Support Response Division (OSE)

https://github.com/wfp-ose/sparc2.git

Contributing

You can get started with geodash-devops.

We're happy to accept pull requests. Please make pull requests against the relevant repo, as listed below.

Repos

  • geodash.github.io : This website
  • geodash-base : The core front-end framework code
  • geodash-framework-django : Back-end Django code. Used by multiple django projects, but doesn't do a lot.
  • geodash-server : Implementation of GeoDash. Demo at http://demo.geodash.io/dashboard/test
  • geodash-devops : DevOps tools for developing GeoDash.
